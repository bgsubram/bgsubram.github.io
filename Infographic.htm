Infographic creation using venngage

While looking for opportunities as an UX Designer, I came across this very interesting Job Description of Venngage.

​

To qualify for the interview, we had to create an Infographic using Venngage.

​

​

Goal:


Browse through the Venngage website and create an infographic explaining how the User Experience can be improved for the website.​

Design Approach:​

​

Since it was my first job search as a UX designer, I created myself a time limit to complete this activity.

​

I saw this job opening on a Sunday evening  and aimed at completion of the infographic within 2-4 hours.

​

Tasks to accomplish:

  1.  Learn how to use Venngage

  2.  Browse through the website

  3.  Look for things that would help the user have a better experience

  4.  Display it in a beautiful infographic

Challenge:

Venngage website already caters a great User Experience.

Being a first time user myself I had to play both the roles of the user and UX designer at the same time!

​

I immediately jotted down some ideas that would improve the website and user experience on paper and started to think how to put it on an infographic.

As i browsed through the website, took screenshots of wherever the change needed to be done and added them to frames in Figma and worked on the changes

BEFORE

FINAL DESIGNING STEPS:

​

  • I took screenshots of the screens I wanted to make a change to

  • Using Figma, added the changes I wanted to do on each of the screen

  • Downloaded them and looked for templates in Venngage

​

Tried few templates

USERS OF VENNGAGE:

Someone who wants to advertise their product, presumably they want quick results with maximum understanding of Vennage as they navigate through the website. The four points I highlighted would help the user to get the maximum information of Venngage.

​

  • I wanted it to be colorful and informative conveying what I intended to say

  • Finally found a very nice template wherein I could fit in the information
